# frightnightcoding2025
Hyperskill Fright Night Coding 2025

## Build and Run

This project uses Gradle with Kotlin.

### Prerequisites
- Java 17 or higher

### Build the project
```bash
./gradlew build
```

### Run the application
```bash
./gradlew run
```

### Run tests
```bash
./gradlew test
```

### Clean build artifacts
```bash
./gradlew clean
```
